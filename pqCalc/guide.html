PQ-TLS Calculator – Instructions

How to use the Post-Quantum TLS Handshake Size Configurator

Overview

This calculator lets you compare the size of cryptographic objects exchanged within a TLS 1.3 handshake. It covers key exchange, signatures, OCSP stapling, Encrypted Client Hello, and Certificate Transparency.

The key exchange can be configured with classical, post-quantum, or hybrid schemes. Available signature schemes are sourced from the PQ-Zoo project.

Creating a Configuration

  1. Key Exchange – Select a scheme from the upper dropdown. Once chosen, pick the desired parameters from the second dropdown. Enable Use hybrid to configure two separate scheme/parameter pairs (KEM + PKI).
  2. Signature Scheme – Choose the signature algorithm and its parameter set from the corresponding dropdowns.
  3. Extensions – Toggle optional extensions on or off:
    • OCSP Stapling – Adds an OCSP response to the server message. Signature scheme is selectable.
    • Certificate Transparency – Adds the configured number of SCTs to the server message.
    • Encrypted Client Hello – Effectively doubles the size of the Client Hello's public key.
  4. Click Add to Comparison to add the configuration to the results list.

Viewing Configurations

Each created configuration appears in the panel on the right, identified by its key-exchange and signature scheme. The total size of transmitted crypto objects is shown at a glance.

Click on a configuration to expand it and reveal detailed breakdown:

  1. Key-exchange scheme name (links to external info), selected parameters, and NIST security level.
  2. Signature scheme name (links to external info), selected parameters, and NIST security level.
  3. Client Hello – Size and type of each transmitted crypto object, plus total size.
  4. Server Hello – Size and type of each transmitted crypto object, plus total size.
  5. Combined handshake size – Sum of all transmitted crypto objects.

Known Limitations & Upcoming Features

Not yet implemented:
  • Selecting which certificate is used with Certificate Transparency.
  • Deleting individual configurations — reload the page to clear all.
  • Importing and exporting created configurations.
